# Setup Overview

DevBrain can be deployed using Docker Compose (recommended) or run locally with Python. Both approaches require the same prerequisites.

## Prerequisites

- **Python 3.11+** (for local setup)
- **Docker & Docker Compose** (for containerized setup)
- **Redis** (local setup only; Docker includes it)
- **OpenAI API key** (or any Cognee-supported LLM provider)
- **GitHub PAT** with `repo` read and webhook access

## Environment Variables

Copy `.env.example` to `.env` and configure:

| Variable | Description |
|----------|-------------|
| `OPENAI_API_KEY` | LLM provider API key |
| `GITHUB_TOKEN` | GitHub PAT with repo + webhook scope |
| `GITHUB_REPO` | Default repo to ingest (`owner/repo`) |
| `GITHUB_WEBHOOK_SECRET` | Secret for webhook signature verification |
| `COGNEE_DATA_DIR` | Local Cognee data directory |
| `COGNEE_SYSTEM_DIR` | Local Cognee system directory |

## Choosing an Approach

- **Docker** - one command, zero dependency management, includes Redis
- **Local** - full control, requires manual Redis + worker process management

See the [Docker](./docker) and [Local](./local) pages for detailed instructions.
