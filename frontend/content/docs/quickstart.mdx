title: Quickstart

---

Get DevBrain running and querying your codebase's history in minutes.

<Steps>
  <Step>

### Configure Environment

```bash
cp .env.example .env
```

Edit `.env` with your secrets. Choose the tab that matches your deployment mode:

```bash tab="Cognee Cloud"
# Cognee Cloud - hosted stores, no local vector/graph infra needed
COGNEE_BASE_URL=https://your-instance.cognee.ai
COGNEE_API_KEY=your_cognee_api_key

# GitHub access (both modes)
GITHUB_TOKEN=ghp_...            # GitHub PAT with repo access
GITHUB_REPO=owner/your-repo     # Default repo to ingest
GITHUB_WEBHOOK_SECRET=...       # Webhook signature secret
```

```bash tab="Local Mode"
# Local mode - file-based Kuzu + LanceDB + SQLite, LLM via Gemini
GEMINI_API_KEY=...              # Get one at aistudio.google.com/apikey

# GitHub access (both modes)
GITHUB_TOKEN=ghp_...            # GitHub PAT with repo access
GITHUB_REPO=owner/your-repo     # Default repo to ingest
GITHUB_WEBHOOK_SECRET=...       # Webhook signature secret
```

  </Step>
  <Step>

### Start Services

```bash
docker compose up --build
```

This starts Redis (job queue), DevBrain API (port 8000), and a background worker for ingestion.

<Callout type="info">
Add `--profile mcp` to also start the MCP server for AI agent connections
(Claude Code, Cursor, OpenCode, etc.).
</Callout>

  </Step>
  <Step>

### Ingest Repository History

```bash
curl -X POST "http://localhost:8000/ingest" \
  -H "Content-Type: application/json" \
  -d '{"repo": "owner/your-repo", "sync_history_days": 90}'
```

  </Step>
  <Step>

### Ask Questions

```bash
curl "http://localhost:8000/query?q=why+was+auth+refactored&repo=owner/your-repo"
```

That's it. DevBrain is now indexing your codebase's history and decisions into a queryable knowledge graph.

  </Step>
</Steps>
