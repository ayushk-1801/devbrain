## Full Historical Sync

```http
POST /ingest
```

Initiates a complete sync of a repository's history - commits, PRs, AST, ADRs, releases.

### Request Body

```json
{
  "repo": "owner/repo-name",
  "sync_history_days": 90
}
```

<TypeTable
  type={{
    repo: { type: "string", description: "Repository to ingest (required)" },
    sync_history_days: { type: "integer", description: "How far back to sync in days. Omit for all history." },
  }}
/>

### Response

```json
{
  "job_id": "abc123",
  "status": "queued",
  "repo": "owner/repo-name"
}
```

## Check Job Status

```http
GET /jobs/{job_id}
```

```json
{
  "job_id": "abc123",
  "status": "in_progress|complete|not_found",
  "result": { }
}
```

## Webhook Receiver

```http
POST /webhook/github
```

Receives GitHub push and pull_request events for incremental ingestion. Includes signature verification via `GITHUB_WEBHOOK_SECRET`.

## Module Pruning

```http
DELETE /module/{repo}/{module}
```

<Callout type="warning" title="Danger">
Never call `forget` without a specific module name - it will delete the entire dataset. This operation is destructive and irreversible.
</Callout>

Surgically removes a deprecated module's subgraph from the knowledge graph.
