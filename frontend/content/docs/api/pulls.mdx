Manage GitHub pull requests directly through DevBrain.

## Create a PR

```http
POST /pulls
```

```json
{
  "repo": "owner/repo-name",
  "title": "Fix login on Safari",
  "head": "fix/safari-login",
  "base": "main",
  "body": "## Summary\nFixes the Safari rendering issue...",
  "draft": false
}
```

## List PRs

```http
GET /pulls?repo=owner/repo-name&state=open
```

## Merge a PR

```http
POST /pulls/{number}/merge
```

```json
{
  "merge_method": "squash"
}
```

Available methods: `merge`, `squash`, `rebase`.

## Update / Close

```http
PATCH /pulls/{number}
```

```json
{
  "title": "Updated title",
  "state": "closed"
}
```

## Close / Reopen

```http
POST /pulls/{number}/close
POST /pulls/{number}/reopen
```

<Callout type="info" title="Authentication">
Write actions require an `X-GitHub-User-Token` header with your own GitHub PAT.
See the [agent setup docs](../agents/claude-code) for details.
</Callout>
