[Antigravity](https://antigravity.google) is Google's agent-first IDE, powered by Gemini. It supports MCP servers through a shared configuration format across the IDE, CLI, and SDK.

## Setup

Edit the global config at `~/.gemini/config/mcp_config.json` or workspace-level `.agents/mcp_config.json`:

```json tab="Docker"
{
  "mcpServers": {
    "devbrain": {
      "command": "docker",
      "args": [
        "run", "-i", "--rm",
        "-e", "DEVBRAIN_API_URL=http://host.docker.internal:8000",
        "devbrain",
        "python", "-m", "backend.mcp_server"
      ]
    }
  }
}
```

```json tab="Python"
{
  "mcpServers": {
    "devbrain": {
      "command": "python",
      "args": ["-m", "backend.mcp_server"],
      "env": {
        "DEVBRAIN_API_URL": "http://localhost:8000"
      }
    }
  }
}
```

### Via IDE Panel

In the Antigravity IDE, click **...** → **MCP Servers** → **Manage MCP Servers** → **View raw config**, then paste the JSON above into the editor.

### Via CLI

In the Antigravity CLI, type `/mcp` to open the interactive MCP Manager overlay and add the server there.

<Callout type="tip" title="Write Actions">
Add `GITHUB_USER_TOKEN` to the `env` block for attributed write actions (creating issues, commenting, etc.).
</Callout>
