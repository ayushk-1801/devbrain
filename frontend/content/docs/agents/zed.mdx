[Zed](https://zed.dev) is a high-performance, multiplayer code editor. It supports MCP servers through the `context_servers` setting.

## Setup

Add to your Zed settings file at `~/.config/zed/settings.json` (or open Settings → Open Settings):

```json tab="Docker"
{
  "context_servers": {
    "devbrain": {
      "command": "docker",
      "args": [
        "run", "-i", "--rm",
        "-e", "DEVBRAIN_API_URL=http://host.docker.internal:8000",
        "devbrain",
        "python", "-m", "backend.mcp_server"
      ]
    }
  }
}
```

```json tab="Python"
{
  "context_servers": {
    "devbrain": {
      "command": "python",
      "args": ["-m", "backend.mcp_server"],
      "env": {
        "DEVBRAIN_API_URL": "http://localhost:8000"
      }
    }
  }
}
```

Reload settings or restart Zed. The DevBrain tools become available to the Zed Agent automatically.

<Callout type="tip" title="Write Actions">
Add `GITHUB_USER_TOKEN` to the `env` block for attributed write actions (creating issues, commenting, etc.).
</Callout>
