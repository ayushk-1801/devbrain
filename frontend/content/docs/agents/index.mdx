DevBrain exposes a Cognee MCP server that any MCP-compatible agent (Claude Code, Cursor, OpenCode) can connect to. Once connected, the agent can query your codebase's full decision history.

## How It Works

The Cognee MCP server runs on a dedicated port and exposes DevBrain's `recall()` capability as MCP tools. Agents call these tools automatically during their sessions to understand the context behind code changes.
